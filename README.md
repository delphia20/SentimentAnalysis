# Sentiment Analysis on Customer Reviews

## Project Overview

This project was developed as part of the CodeAlpha Data Analytics Internship.

The objective of this project is to analyze customer reviews and classify them into Positive, Negative, or Neutral sentiments using Natural Language Processing (NLP) techniques.

## Technologies Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Jupyter Notebook

## Project Objectives

* Analyze textual review data.
* Determine customer sentiment using sentiment polarity.
* Classify reviews into Positive, Negative, and Neutral categories.
* Visualize sentiment distribution using charts.

## Dataset Information

The dataset consists of customer reviews collected for sentiment analysis.

Columns Used:

* Review
* Sentiment

## Methodology

### Data Collection

* Loaded customer review data into a Pandas DataFrame.

### Text Processing

* Processed review text using TextBlob.

### Sentiment Classification

* Positive Sentiment (Polarity > 0)
* Negative Sentiment (Polarity < 0)
* Neutral Sentiment (Polarity = 0)

### Data Visualization

* Bar Chart
* Pie Chart

## Results

The sentiment analysis successfully categorized reviews into:

* Positive Reviews
* Negative Reviews
* Neutral Reviews

Visualizations were generated to understand the distribution of customer opinions.
